== Description ==

Adds Future scheduled posts to preview pages, so you can preview what your site will look like before the post goes live. 

== Installation ==

1. Upload `ss_homepage_preview` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click the Preview Link on a page, OR add `?preview=true` to the page's URL manually
